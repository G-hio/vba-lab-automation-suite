Automatización de flujos de trabajo desarrollada para la gestión de inventarios y facturación técnica. Esta solución optimiza la entrada de datos críticos en el entorno de laboratorio.

###  Detalles del Proyecto
* **Normalización de Datos:** Procesamiento automático de prefijos de áreas (QH, QICL, AAMT).
* **Generación de Parámetros:** Inserción automatizada de ítems para Análisis Proximal.
* **Optimización:** Reducción de tiempos de digitación y eliminación de errores manuales.
